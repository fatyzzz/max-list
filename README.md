# max-cleaner

Списки доменов, связанных с мессенджером MAX.

- `ip-check.txt` — сервисы определения IP-адреса (домен + путь)
- `ip-check-formatted.txt` — то же, только домены без путей
- `vpndetect.txt` — домены для скрытой проверки доступности сервисов

Обновляется ежедневно через GitHub Actions.

---

Данный репозиторий создан исключительно в ознакомительных и исследовательских целях. Автор не совершает противоправных действий и не призывает к ним. Вся информация получена из открытых источников.
